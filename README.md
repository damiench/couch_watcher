# couch_watcher


[![node][node]][node-url] ![yarn][yarn] ![npm][npm] ![docker][docker]

This app monitors changes in your couchDB base and writes it in AWS S3.

### To run app:
```sh
npm i
# or
yarn

npm start
```

#### ! before start fill all fields !
```sh
settings-aws.json
settings.json
```
